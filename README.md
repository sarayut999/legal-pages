# legal-pages

Shared GitHub Pages host for privacy policy / support pages across the app portfolio.
App repos stay private; this repo is public so Pages can serve without paid GitHub.

Add a new app: create `<app>/privacy-policy.html` + `<app>/support.html`, add a link in
`index.html`, commit, push. Live at `https://sarayut999.github.io/legal-pages/<app>/...`.
